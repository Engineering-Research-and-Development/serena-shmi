# serena-shmi

### Getting Started

Clone repo

  git clone https://github.com/Engineering-Research-and-Development/serena-shmi

Explore path

  $> cd serena-shmi

Install serena-shmi modules

  npm install

Install modules of shmi-mdinterface,shmi-mdmanager,shmi-webserver,shmi-dashboard

  npm run install

Run all services

  npm run start
