# serena-shmi

### Getting Started
1) Clone repo

...
git clone https://github.com/Engineering-Research-and-Development/serena-shmi
...

2) Explore path

...
$> cd serena-shmi
...

3) Install serena-shmi modules

...
npm install
...

4) Install modules of shmi-mdinterface,shmi-mdmanager,shmi-webserver,shmi-dashboard

...
npm run install
...

5) run all services

...
npm run start
...
